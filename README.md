# Hand Recognition LED System

## Project Overview
This Engineering project is a touchless help system built on a Raspberry Pi. It uses a camera and MediaPipe hand tracking to detect gestures and trigger different outputs.

## Features
- Open palm held for 3 seconds triggers a caregiver alert
- Two finger gesture turns on lights
- Waving gesture triggers an emergency alert
- GPIO controlled LEDs and buzzer
- Optional caregiver email alert
- On screen feedback including gesture state, hand status, and FPS

## Hardware
- Raspberry Pi
- USB camera
- LEDs
- Buzzer
- Breadboard and jumper wires

## Software
- Python
- OpenCV
- MediaPipe
- gpiozero
